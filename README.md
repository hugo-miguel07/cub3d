# cub3D

## Description

**cub3D** is a 3D graphics project from the 42 curriculum inspired by *Wolfenstein 3D*. The goal is to build a simple first-person game engine using the **raycasting** technique and the **MiniLibX** library.

The program parses a `.cub` map file, validates its contents, and renders a pseudo-3D environment where the player can move and explore the map.

## Instructions

### Compilation

```bash
make
```

### Execution

```bash
./cub3D <map.cub>
```

### Controls

* **W, A, S, D**: Move
* **← / →**: Rotate camera
* **ESC**: Exit
